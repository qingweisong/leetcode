问题：
```
给你一个整数 n，请你返回 任意 一个由 n 个 各不相同 的整数组成的数组，并且这 n 个数相加和为 0 。(1 <= n <= 1000)
```
示例：
```
输入：n = 5
输出：[0,1,2,3,-6]
解释：这些数组也是正确的 [-5,-1,1,2,3]，[-3,-1,2,-2,4]。

输入：n = 3
输出：[-1,0,1]

输入：n = 1
输出：[0]
```
思路：
```
先将 n - 1 个自然数 0, 1, 2, ..., n - 2 放入数组中，它们的和为 sum。
对于剩下的 1 个数，令其为 -sum，此时这 n 个数的和为 0
```
代码：
```python
class Solution:
    def sumZero(self, n: int) -> List[int]:
        ans = [x for x in range(n - 1)]
        ans.append(-sum(ans))
        return ans
```
